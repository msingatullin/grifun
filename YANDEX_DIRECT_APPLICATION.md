# Заявка на предоставление полного доступа к API Яндекс.Директа

## 1. Какие новые возможности работы с Директом дает ваше приложение пользователям?*

**Ответ:**

Наше приложение предоставляет автоматическую AI-оптимизацию рекламных кампаний в Яндекс.Директе, что позволяет:

- **Интеллектуальный анализ эффективности кампаний**: Приложение использует искусственный интеллект (OpenAI GPT-4) для глубокого анализа статистики кампаний, выявления проблем и автоматической генерации конкретных рекомендаций по улучшению.

- **Автоматическая оптимизация без ручного вмешательства**: Система самостоятельно анализирует CTR, стоимость клика, конверсию и другие метрики, предлагая оптимальные изменения ставок, ключевых слов и текстов объявлений.

- **Проактивные рекомендации с приоритизацией**: AI не просто показывает статистику, а предлагает конкретные действия с указанием приоритета (высокий/средний/низкий) и обоснованием каждого изменения.

- **Непрерывное улучшение кампаний**: Автоматическая оптимизация по расписанию (cron) позволяет постоянно улучшать эффективность кампаний без необходимости ежедневного мониторинга.

- **Снижение стоимости лида**: Система фокусируется на оптимизации для снижения стоимости привлечения клиента при сохранении или увеличении конверсии.

- **Экономия времени рекламодателя**: Вместо ручного анализа данных и принятия решений, рекламодатель получает готовые рекомендации, которые можно сразу применять.

## 2. Опишите схему взаимодействия вашего приложения с Директом*

**Ответ:**

Схема взаимодействия приложения с API Яндекс.Директа:

**1. Получение данных о кампаниях:**
   - Приложение использует метод `campaigns.get` для получения списка всех активных кампаний
   - Извлекает информацию о статусе, бюджете, настройках кампаний

**2. Сбор статистики:**
   - Использует Reports API (`reports`) для получения детальной статистики по кампаниям за заданный период
   - Анализирует метрики: показы, клики, CTR, стоимость клика, конверсии

**3. AI-анализ и оптимизация:**
   - Передает собранные данные в AI-модель (OpenAI GPT-4o-mini)
   - AI анализирует эффективность и генерирует рекомендации:
     * Оценка эффективности (score 0-100)
     * Краткое резюме проблем и возможностей
     * Список конкретных рекомендаций
     * Предложенные изменения с указанием типа (ставки, ключевые слова, тексты объявлений, бюджет, таргетинг)

**4. Логирование и отчетность:**
   - Все результаты анализа сохраняются в логах
   - Формируются структурированные отчеты для дальнейшего анализа

**5. Автоматическое выполнение (планируется):**
   - Приложение может автоматически применять рекомендуемые изменения через методы `campaigns.update`, `keywords.update`, `ads.update`
   - Все изменения логируются для контроля

**Технические детали:**
- Протокол: JSON
- Методы API: campaigns.get, reports, campaigns.update (планируется)
- Частота запросов: согласно лимитам API Яндекс.Директа
- Авторизация: OAuth 2.0 токен

## 3. Спецификации, скриншоты интерфейса*

**Описание файлов:**

Приложение представляет собой backend-модуль для Next.js с REST API endpoints. Основной функционал доступен через HTTP API, без веб-интерфейса.

**Основные endpoints:**
- `GET /api/yandex-direct/campaigns` - получение списка кампаний
- `POST /api/yandex-direct/campaigns/optimize` - оптимизация кампаний с AI
- `POST /api/yandex-direct/stats` - получение статистики
- `POST /api/yandex-direct/auto-optimize` - автоматическая оптимизация

**Пример ответа API:**

```json
{
  "success": true,
  "campaignsAnalyzed": 5,
  "optimizations": [
    {
      "campaignId": 12345,
      "campaignName": "Название кампании",
      "score": 75,
      "summary": "Кампания показывает хорошие результаты...",
      "recommendations": [
        "Увеличить ставки на высокочастотные запросы",
        "Добавить минус-слова для снижения нерелевантного трафика"
      ],
      "suggestedChanges": [
        {
          "type": "bid",
          "action": "Увеличить ставку на 20%",
          "reason": "Низкий CTR при хорошей конверсии",
          "priority": "high"
        }
      ]
    }
  ]
}
```

**Архитектура:**
- Язык: TypeScript/Node.js
- Фреймворк: Next.js 14 (App Router)
- AI: OpenAI GPT-4o-mini
- Протокол взаимодействия: JSON через HTTPS

## 4. По возможности предоставьте нам демо-доступ к программе

**Ответ:**

Демо-доступ к API:
- URL: `https://grifun.ru/api/yandex-direct/campaigns`
- Метод: GET (для получения списка кампаний)
- Авторизация: не требуется для демо (или можно предоставить тестовый токен)

Для полного доступа к функционалу оптимизации:
- Endpoint: `https://grifun.ru/api/yandex-direct/campaigns/optimize`
- Метод: POST
- Требуется: OAuth токен Яндекс.Директа

**Документация API:**
- Полная документация доступна в репозитории проекта
- README с примерами использования: `/lib/yandex-direct/README.md`
