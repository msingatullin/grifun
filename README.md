# AI Automation Agency - Landing Page

Профессиональный лендинг для B2B SaaS агентства по автоматизации бизнеса с помощью искусственного интеллекта.

## Технологии

- **Next.js 14+** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (плавные профессиональные анимации)
- **Lucide React** (иконки)

## Дизайн

- **Стиль**: Премиум B2B SaaS (вдохновлено Linear, Stripe, Raycast)
- **Цветовая схема**: Темный фон (#0f172a), акценты синий/индиго
- **Типографика**: Inter (профессиональный sans-serif)
- **Анимации**: Плавные, сдержанные переходы (ease-out)

## Структура проекта

```
├── app/
│   ├── layout.tsx           # Корневой layout
│   ├── page.tsx             # Главная страница
│   └── globals.css          # Глобальные стили
├── components/
│   ├── Header.tsx           # Навигация
│   ├── Hero.tsx            # Главный экран
│   ├── ValueProposition.tsx # Преимущества автоматизации
│   ├── Services.tsx        # Решения (Bento Grid)
│   ├── Process.tsx        # Процесс работы (Timeline)
│   └── Footer.tsx         # Футер
└── ...
```

## Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен-версии
npm start
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

- **Профессиональный тон**: Серьезный, бизнес-ориентированный контент
