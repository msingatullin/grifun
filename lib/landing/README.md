# Генератор Landing Pages

Система автоматической генерации персонализированных лендингов на основе запроса пользователя.

## Использование

### Через URL с query параметром:
```
/lp?query=автоматизация%20продаж
```

### Через slug:
```
/lp/avtomatizatsiya-prodazh
```

## Архитектура

1. **normalizeQuery** - нормализует запрос пользователя
2. **makeLandingSpec** - генерирует структуру лендинга через OpenAI
3. **renderLanding** - рендерит лендинг по универсальному шаблону
4. **Кэширование** - сохраняет спек в `data/landings/{slug}.json` с TTL 7 дней

## Guardrails

- Удаление гарантий ("гарантируем", "100%", "99%")
- Нейтральные формулировки для кейсов
- Автоматические дисклеймеры
- Уровни compliance: "strict" (для госсектора/финансов) и "normal"

## Админ-функции

Для регенерации лендинга добавьте параметр:
```
/lp/{slug}?regenerate={ADMIN_SECRET_KEY}&query={query}
```

Установите `ADMIN_SECRET_KEY` в `.env.local`
