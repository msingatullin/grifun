# Документы

Здесь должны быть размещены PDF файлы документов.

Формат имен файлов:
- contract-template_v1.2_2025-12-19.pdf
- sla_v1.0_2025-12-19.pdf
- privacy_v1.0_2025-12-19.pdf
- implementation-regulation_v1.1_2025-12-19.pdf
- nda-template_v1.0_2025-12-19.pdf
- roi-method_v1.0_2025-12-19.pdf

При обновлении документов:
1. Замените PDF файл с новым именем (обновите версию и дату)
2. Обновите data/docs.json с новым путем, версией и датой

ВАЖНО: Папка переименована в /documents/ (вместо /docs/) чтобы избежать конфликта с роутом /docs
