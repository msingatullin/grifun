# Исправление проблемы с установкой бюджета

## Проблема

При попытке установить бюджет через API возникает ошибка:
- "Omitted required parameter Mode" - исправлено, добавлен Mode
- "Inconsistent object state" или "Field set incorrectly" - требуется дополнительная настройка

## Возможные причины

1. **Перфоманс-кампании** могут требовать другой формат бюджета
2. Нужно получить текущее состояние кампании перед обновлением
3. Может требоваться указание дополнительных полей

## Временное решение

Пока автоматическое применение бюджета дорабатывается, используйте:

### Вариант 1: Вручную через интерфейс Яндекс.Директа
1. Зайдите в Яндекс.Директ
2. Откройте кампанию 705271290
3. Установите дневной бюджет: 5000 руб/день

### Вариант 2: Через API с полными данными кампании
Нужно сначала получить полную информацию о кампании, затем обновить.
